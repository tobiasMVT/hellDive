# HellDive - Tuning Notes

## Bonus Free Spin / Free Respin
- Free-spin and free-respin waves should display a red hell-energy wave over the tiles.
- Tiles should pulse/grow when the wave passes.
- Demon spawns should feel like the demon is being added to that exact position, with portal/fire impact feedback.

## Combat Feel
- The angel should feel powerful and readable.
- Charge-ups should have anticipation before impact.
- Kills should have a clear impact, splash, and loot payoff.

## Divine Ground Feedback
When Divine Wrath affects positions, those cells should briefly remain lit with divine power so the player understands which ground area was affected.

## Prototype Asset Pass
- New generated assets live in `src/game-client/assets/helldive/`.
- Main game background is Heaven City: blue sky, white marble, gold divine details, portal in sky.
- Bonus background is Hell floor: dark stone/lava, rear portal, divine pentagram landing zone.
- Symbols 1-8 are now divine marble/gold symbols instead of fruit placeholders.
- Demon symbols now use imp/brute/boss art. Symbol 13 is treated visually as a 3x3 boss demon.
- Hero/monkey art now points to female angel placeholder stages.
- Loot drops now use coin/jewel images instead of colored dots.
- Red wave, red portal spawn, divine ground, angel trail, and demon splash effects have image assets.
