# HellDive - Game Summary

## Concept
HellDive is about demons invading heaven. The main game takes place in a heaven city with white marble, blue sky, golden divine details, and a demon portal that opens wider as the user kills demons.

## Current Placeholder Mapping
- Banana = demon
- Monkey = female angel
- Fruit symbols = white marble / divine heaven symbols for now

## Core Loop
Each demon killed in the main game has a chance to trigger the bonus. The selling point is the visual combat system: the angel should fly around, attack demons, and make kills feel impactful. Demons can drop loot.

## Bonus
When the bonus triggers, the angel performs a HellDive into the portal and lands in hell. Hell uses Diablo 2 inspired dark stone, fire, lava details, and a portal behind the game area. The angel lands in the center of an angel pentagram on the dark stone floor.

The center position is off limits and displays accumulated multipliers.

## Bonus Abilities
- Portal: injects demons before each free-spin wave. Summoned demons should appear in a cluster pattern.
- Divine Wrath: before an attack, the angel charges up and strikes. Server-provided affected positions light up with divine power. Demons in that area are instantly killed and guaranteed to drop loot.
- Divine Strike / Divine Charge: randomly triggers when the angel is about to move to a demon. The angel stays in place, charges, then rushes toward the target with a Matrix-style angel trail. The demon should explode/splash heavily.

## Loot
Loot drops are coins and different jewels.

## Boss Demon
The boss demon is huge and covers 3x3.

## Art Direction
Use Diablo 2 as inspiration, but not cute baby angels or halo-style angels. Heaven should feel like a divine marble city. Hell should feel dark, heavy, stone, lava, fire, and demonic.
